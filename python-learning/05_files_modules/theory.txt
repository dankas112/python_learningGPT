МОДУЛИ в Python — самое важное
Что такое модуль?

Это обычный .py файл, который можно подключить в другом файле через import.

Например:

Файл math_utils.py:

def add(a, b):
    return a + b


Файл main.py:

import math_utils

print(math_utils.add(3, 4))

Варианты импорта
1) Обычный импорт
import math
print(math.sqrt(16))

2) Импорт модуля с переименованием
import math as m
print(m.sqrt(16))

3) Импорт конкретных функций
from math import sqrt, ceil
print(sqrt(25))

4) Импорт всего, но так делать НЕ надо:
from math import *


Это плохая практика, потому что засоряет пространство имён.

Как устроена структура проектов
project/
│
├── main.py
├── utils.py
└── helpers/
      ├── __init__.py
      └── file_tools.py


Любой файл можно импортировать в другой:

from helpers.file_tools import save_file

Специальная переменная __name__

Позволяет делать «запуск функции только если файл запущен как программа».

Пример:

math_utils.py:

def add(a, b):
    return a + b

if __name__ == "__main__":
    print("Тест:", add(2, 3))


Если импортировать — тест не запустится.
Если запустить файл напрямую — тест появится.
