1. УСЛОВИЯ (if, elif, else)
Позволяют выполнять код только если условие истинно.
    if условие:
        действия_если_истинно
    elif другое_условие:
        действия_если_второе_истинно
    else:
        действия_если_всё_ложь
Примеры:
    x = 10
    if x > 5:
        print("Больше 5")
    elif x == 5:
        print("Равно 5")
    else:
        print("Меньше 5")
Логические операторы:
        and — логическое И
        or — логическое ИЛИ
        not — логическое НЕ
    if age > 18 and age < 60:
    print("Работающий возраст")
Вложенные условия:
    if x > 0:
        if x < 10:
            print("От 1 до 9")
Тернарный оператор:
    status = "совершеннолетний" if age >= 18 else "несовершеннолетний"
2. ЦИКЛ for
Предназначен для перебора элементов.
Пример с range():
    for i in range(5):  # от 0 до 4
        print(i)
Пример по диапазону:
    for i in range(1, 11):
        print(f"{i} * 2 = {i * 2}")
Перебор строк и списков:
    for letter in "python":
        print(letter)

    for item in [1, 2, 3]:
        print(item)
3. ЦИКЛ while
Выполняется, пока условие истинно.
    x = 0
    while x < 5:
        print(x)
        x += 1
Важно: всегда должно быть условие выхода, иначе получится бесконечный цикл.
 4. break, continue, pass
Команда | Что делает
break | Прерывает цикл полностью
continue | Переходит к следующей итерации цикла
pass | Ничего не делает (заглушка)
    
for i in range(10):
    if i == 5:
        break
    print(i)  # Выведет 0-4

for i in range(5):
    if i == 2:
        continue
    print(i)  # Пропустит 2

if True:
    pass  # Временно ничего
