1. –°—Ç—Ä–æ–∫–∏ (str)

–°—Ç—Ä–æ–∫–∞ ‚Äî —ç—Ç–æ –ø–æ—Å–ª–µ–¥–æ–≤–∞—Ç–µ–ª—å–Ω–æ—Å—Ç—å —Å–∏–º–≤–æ–ª–æ–≤.

üìå –°–æ–∑–¥–∞–Ω–∏–µ –∏ –∏–Ω–¥–µ–∫—Å–∞—Ü–∏—è:
text = "Python"
print(text[0])  # P
print(text[-1]) # n

üîπ –°—Ä–µ–∑—ã:
print(text[0:4])  # 'Pyth' ‚Äî —Å–∏–º–≤–æ–ª—ã —Å 0 –ø–æ 3
print(text[2:])   # 'thon' ‚Äî –æ—Ç 2 –¥–æ –∫–æ–Ω—Ü–∞
print(text[:3])   # 'Pyt' ‚Äî —Å –Ω–∞—á–∞–ª–∞ –¥–æ 2
print(text[::-1]) # 'nohtyP' ‚Äî –ø–µ—Ä–µ–≤–æ—Ä–æ—Ç —Å—Ç—Ä–æ–∫–∏

üîπ –û—Å–Ω–æ–≤–Ω—ã–µ –º–µ—Ç–æ–¥—ã —Å—Ç—Ä–æ–∫:
s = "  hello world  "
print(s.upper())     # HELLO WORLD
print(s.lower())     # hello world
print(s.strip())     # 'hello world' ‚Äî —É–±–∏—Ä–∞–µ—Ç –ø—Ä–æ–±–µ–ª—ã —Å –Ω–∞—á–∞–ª–∞ –∏ –∫–æ–Ω—Ü–∞
print(s.split())     # ['hello', 'world'] ‚Äî —Ä–∞–∑–±–∏–≤–∞–µ—Ç –Ω–∞ —Å–ø–∏—Å–æ–∫ –ø–æ –ø—Ä–æ–±–µ–ª–∞–º
print("-".join(["a","b","c"])) # 'a-b-c' ‚Äî —Å–æ–µ–¥–∏–Ω—è–µ—Ç —ç–ª–µ–º–µ–Ω—Ç—ã —Å–ø–∏—Å–∫–∞
print(s.replace("hello", "hi")) # –∑–∞–º–µ–Ω—è–µ—Ç –ø–æ–¥—Å—Ç—Ä–æ–∫—É
print("world" in s)  # True ‚Äî –ø—Ä–æ–≤–µ—Ä–∫–∞ –Ω–∞–ª–∏—á–∏—è –ø–æ–¥—Å—Ç—Ä–æ–∫–∏

üîπ 2. –°–ø–∏—Å–∫–∏ (list)

–°–ø–∏—Å–æ–∫ ‚Äî –∏–∑–º–µ–Ω—è–µ–º–∞—è –ø–æ—Å–ª–µ–¥–æ–≤–∞—Ç–µ–ª—å–Ω–æ—Å—Ç—å —ç–ª–µ–º–µ–Ω—Ç–æ–≤.

üìå –°–æ–∑–¥–∞–Ω–∏–µ:

üîπ –ò–Ω–¥–µ–∫—Å–∞—Ü–∏—è –∏ —Å—Ä–µ–∑—ã:

print(numbers[-1])  # 5

üîπ –ú–µ—Ç–æ–¥—ã —Å–ø–∏—Å–∫–æ–≤:
numbers.append(6)       # –¥–æ–±–∞–≤–∏—Ç—å —ç–ª–µ–º–µ–Ω—Ç –≤ –∫–æ–Ω–µ—Ü
numbers.insert(2, 99)   # –≤—Å—Ç–∞–≤–∏—Ç—å 99 –Ω–∞ –ø–æ–∑–∏—Ü–∏—é 2
numbers.remove(3)       # —É–¥–∞–ª–∏—Ç—å –ø–µ—Ä–≤—ã–π —ç–ª–µ–º–µ–Ω—Ç —Å–æ –∑–Ω–∞—á–µ–Ω–∏–µ–º 3
numbers.pop()           # —É–¥–∞–ª–∏—Ç—å –ø–æ—Å–ª–µ–¥–Ω–∏–π —ç–ª–µ–º–µ–Ω—Ç –∏ –≤–µ—Ä–Ω—É—Ç—å –µ–≥–æ
numbers.sort()          # —Å–æ—Ä—Ç–∏—Ä–æ–≤–∫–∞ –ø–æ –≤–æ–∑—Ä–∞—Å—Ç–∞–Ω–∏—é
numbers.reverse()       # –æ–±—Ä–∞—Ç–∏—Ç—å –ø–æ—Ä—è–¥–æ–∫
len(numbers)            # –¥–ª–∏–Ω–∞ —Å–ø–∏—Å–∫–∞

üîπ –ò—Ç–µ—Ä–∞—Ü–∏—è –ø–æ —Å–ø–∏—Å–∫—É:
for num in numbers:
    print(num)

üîπ 3. –í–ª–æ–∂–µ–Ω–Ω—ã–µ —Å–ø–∏—Å–∫–∏ (—Å–ø–∏—Å–∫–∏ –≤–Ω—É—Ç—Ä–∏ —Å–ø–∏—Å–∫–∞)

üîπ –°–æ–≤–µ—Ç—ã

–°—Ç—Ä–æ–∫–∏ –º–æ–∂–Ω–æ –ø—Ä–µ–≤—Ä–∞—â–∞—Ç—å –≤ —Å–ø–∏—Å–∫–∏ —á–µ—Ä–µ–∑ split(), –æ–±—Ä–∞—Ç–Ω–æ —á–µ—Ä–µ–∑ join()
